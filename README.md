# ChrisGVE Homebrew Tap

Custom Homebrew formulae by [ChrisGVE](https://github.com/ChrisGVE).

## Usage

```bash
brew tap ChrisGVE/tap
```

Then install any available formula:

```bash
brew install ChrisGVE/tap/<formula>
```

## Available Formulae

| Formula | Description |
|---------|-------------|
| [workspace-qdrant-mcp](https://github.com/ChrisGVE/workspace-qdrant-mcp) | Project-scoped vector database MCP server with hybrid search |

## Troubleshooting

```bash
brew update
brew doctor
```
